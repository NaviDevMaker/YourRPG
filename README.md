---

## 操作方法

- **W / A / S / D** または **方向キー**：一マスごとに移動（長押しで連続移動可能）  
- **X**：ステータス画面を開く（戦闘中は前の選択肢に戻る）  
- **スペースキー**：選択肢を決定・実行  
- **エンターキー**：タイトル画面での名前決定に使用  
- **G**：ゴールドUIの表示／非表示を切り替え

---

## 実行方法

- `YourRPG.exe` を実行してください。  
※ ビルド済みファイルはギガファイル便にて配布予定です。
